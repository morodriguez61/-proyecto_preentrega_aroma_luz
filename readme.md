
Este es mi proyecto realizado en HTML y CSS para una tienda de velas y productos aromáticos, realizado para el curso de FRONT-END.

## Contenido de la página

- Inicio
- Productos
- Reseñas
- Formulario de contacto
- Redes sociales

## Tecnologías utilizadas

- HTML
- CSS
- Flexbox
- Grid
- Media Queries

## Características

- Diseño responsive
- Barra de navegación
- Cards de productos
- Video de presentación
- Formulario de contacto
